What is this?

A way for you to immortalize your message on the blockchain.
Write whatever you want.
A smart contract instantly stores the newest message on the blockchain, if your bid is highest.
This website displays the latest message.
Nobody can stop it or turn it off, including me!

How does it work?

Running on the ethereum blockcain is the smart contract with address which compares every incoming transaction with previous maximum value. If your value is higher, smart contract stores your message in the blockchain.
The previous message is still in the blockchain, but only the last one is displayed on the website.

What if I send insufficient funds/somebody raises the top price seconds before me?

Smart contract refunds you.

Yes. Smart contract source code is here.
